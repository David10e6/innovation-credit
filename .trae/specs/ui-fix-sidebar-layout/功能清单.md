# 各端已有功能清单

## 学生端

| 功能名称 | 功能路径 / 入口 | 核心操作逻辑 |
|---------|----------------|-------------|
| 学院通知 | /student | 查看学院发布的通知信息 |
| 学校通知 | /student/school-notice | 查看学校发布的通知信息 |
| 学分申请 | /student/credit/apply | 填写学分申请表单→提交审核→等待审核结果 |
| 学分列表 | /student/credit/list | 查看已获得的学分列表 |
| 申请记录 | /student/apply/list | 查看所有学分申请记录→点击查看详情→可编辑未审核的申请 |
| 申请详情 | /student/apply/detail/:id | 查看单个学分申请的详细信息 |
| 申请编辑 | /student/apply/edit/:id | 编辑未审核的学分申请信息 |
| 活动通知 | /student/activity/notice | 查看活动通知信息 |
| 评分规则 | /student/score/rules | 查看学分评分规则 |

## 辅导员端

| 功能名称 | 功能路径 / 入口 | 核心操作逻辑 |
|---------|----------------|-------------|
| 待审核列表 | /counselor/apply/list | 查看待审核的学分申请→点击审核→通过/驳回申请 |
| 审核详情 | /counselor/apply/audit/:id | 查看单个学分申请的详细信息并进行审核操作 |
| 学生列表 | /counselor/student/list | 查看所属学院的学生列表 |
| 活动管理 | /counselor/activity/management | 管理学院级活动信息 |
| 通知管理 | /counselor/notice/management | 管理学院级通知信息 |

## 管理员端

| 功能名称 | 功能路径 / 入口 | 核心操作逻辑 |
|---------|----------------|-------------|
| 用户管理 | /admin/user/list | 管理系统用户信息 |
| 学分统计 | /admin/credit/statistics | 查看全校学分统计数据 |
| 活动管理 | /admin/activity/management | 管理校级活动信息 |
| 通知管理 | /admin/notice/management | 管理校级通知信息 |
| 学生管理 | /admin/student/management | 管理全校学生信息 |

## 登录页面

| 功能名称 | 功能路径 / 入口 | 核心操作逻辑 |
|---------|----------------|-------------|
| 用户登录 | /login | 输入用户名/密码→选择角色→提交登录→根据角色跳转对应首页 |
